# Modern ESX Status HUD for FiveM

A sleek and modern status HUD for **FiveM ESX servers**, designed to display player stats like **health, armor, hunger, thirst**, Fully customizable and lightweight, perfect for roleplay servers looking for a clean and functional interface.

---

## Features

- **Health & Armor**: Real-time display of player health and armor status.
- **Hunger & Thirst**: Shows current hunger and thirst levels for survival roleplay.
- **Custom Colors & Gradients**: Fully configurable colors and progress bar styles.
- **Responsive & Modern Design**: Minimalist HUD that looks great on all resolutions.
- **Optimized for Performance**: Lightweight and smooth, with minimal FPS impact.
- **ESX Integration**: Works seamlessly with the ESX framework.

---
